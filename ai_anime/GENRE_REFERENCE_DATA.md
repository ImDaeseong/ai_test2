# Genre Reference Data

## Purpose

`configs/genre_reference_profiles.json` supplements the existing genre classifier with sourced, generalized visual grammar.

It does not replace:

- `configs/genres.json` genre matching
- song lyrics and emotion analysis
- song-specific character generation
- palette and platform formatting rules

## Pilot Families

- rock
- acoustic/ballad
- hip-hop/trap
- electronic/synth
- idol pop
- jazz/soul

Each family contains:

- narrative modes
- environments
- camera language
- motion language
- editing and transition language
- lighting language
- character direction
- visual cliches to avoid

## Provenance

The source table is stored in the config. It uses public material from the Rock & Roll Hall of Fame, Smithsonian Institution, Smithsonian Folkways, Smithsonian museums, GRAMMY Museum, and MoMA.

Source names and URLs are provenance only. The generator receives only generalized visual grammar.

## Rights And Originality

Generated prompts must not reproduce:

- recognizable artists or faces
- copyrighted characters
- logos or readable brand marks
- exact costumes
- signature props
- exact music-video shot sequences
- exact stage designs

Actual generated images and clips still require human originality review.
