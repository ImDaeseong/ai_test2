# 09 Video Motion Prompts

Purpose: 생성된 이미지를 Flow, Veo, Kling 같은 영상 AI에서 움직이는 클립으로 만들 때 사용하는 Motion Prompt.

영상 프롬프트에는 새 캐릭터나 새 세계관을 만들지 않는다. 이미지를 기준으로 카메라 움직임, 연주 동작, 조명, 연기, 관객 반응만 지시한다.

영상화할 업로드 이미지는 README.md에 명시된 기준 이미지(reference 폴더)의 AI Boy/AI Girl 캐릭터 스타일을 참조해 만든 이미지여야 한다.

## CapCut 편집 길이 (Kling 생성 설정 아님)

아래는 CapCut에서 각 클립을 잘라 쓸 때의 권장 활용 길이입니다. Kling UI의 생성 시간 설정과 다릅니다.

```text
vocal: 2-5 seconds per cut
guitar: 3-6 seconds per cut
bass: 3-6 seconds per cut
drum: 2-4 seconds per cut
stage: 5-8 seconds per cut
atmosphere: 5-8 seconds per cut
```

## Kling Duration Settings

Kling ignores duration text inside the prompt. Set the clip length in the Kling UI before generating.

```text
vocal   → Kling UI: 5 seconds
guitar  → Kling UI: 5 seconds
bass    → Kling UI: 5 seconds
drum    → Kling UI: 5 seconds
stage   → Kling UI: 10 seconds
atmosphere → Kling UI: 10 seconds
```

For a 3-minute MV, generate 12 clips total (8 performance at 5s + 4 stage/atmosphere at 10s).
Reuse strong chorus clips 2-3 times in CapCut to cover the full 180 seconds.

## Flow Duration Settings

Flow (Google) ignores duration text inside the prompt. Set the clip length in the Flow UI before generating.
Flow supports 5 seconds or 8 seconds per clip.

```text
vocal   → Flow UI: 5 seconds
guitar  → Flow UI: 5 seconds
bass    → Flow UI: 5 seconds
drum    → Flow UI: 5 seconds
stage   → Flow UI: 8 seconds
atmosphere → Flow UI: 8 seconds
```

## Veo Duration Settings

Veo (Google) partially reads duration text inside the prompt but the UI setting takes priority.
Add duration to the prompt as a hint. Veo supports up to 8 seconds per clip.

```text
vocal   → Veo UI: 5 seconds  (add "5 seconds" at the end of the prompt)
guitar  → Veo UI: 5 seconds  (add "5 seconds" at the end of the prompt)
bass    → Veo UI: 5 seconds  (add "5 seconds" at the end of the prompt)
drum    → Veo UI: 5 seconds  (add "5 seconds" at the end of the prompt)
stage   → Veo UI: 8 seconds  (add "8 seconds" at the end of the prompt)
atmosphere → Veo UI: 8 seconds  (add "8 seconds" at the end of the prompt)
```

Note: Veo stage and atmosphere clips max out at 8 seconds (not 10). For a 3-minute MV this is sufficient.
When using Veo for stage or atmosphere, add the duration hint to the prompt before submitting.

## Kling Production Rule

Use the six core images as source images, then generate multiple motion variants from each image.
Do not ask Kling to create all variants inside one generation. Upload the same image again and run one
motion prompt at a time, or reuse the same image with Variant A, Variant B, and Variant C.

```text
minimum 3-minute MV set:
vocal image: 3 clips
guitar image: 3 clips
bass image: 2 clips
drum image: 2 clips
stage image: 3 clips
atmosphere image: 2 clips
total: 15 clips

recommended 3-minute MV set:
vocal image: 5 clips
guitar image: 5 clips
bass image: 3 clips
drum image: 3 clips
stage image: 5 clips
atmosphere image: 5 clips
total: 26 clips

recommended 4-minute MV set:
vocal image: 8 clips
guitar image: 6 clips
bass image: 4 clips
drum image: 4 clips
stage image: 5 clips
atmosphere image: 5 clips
total: 32 clips
```

For rock, pop, hip-hop, or high-energy songs, the chorus and final chorus should
use faster cuts in CapCut. Verse sections can hold clips for 5 to 8 seconds. Chorus sections can cut
every 1 to 3 seconds by reusing the best Kling clips.

## CapCut Editing Map

Use the best generated clips first. Reuse strong chorus clips intentionally; repeated performance shots
feel natural if they return on chorus hooks.

```text
Intro: stage A, atmosphere B, vocal A
Verse 1: vocal A, bass A, guitar B, drum A
Pre-Chorus: stage B, vocal C
Chorus 1: stage B, vocal B, guitar C, drum B, atmosphere C
Verse 2: vocal C, bass B, guitar A, atmosphere A
Bridge or Solo: guitar A, drum C, stage A
Final Chorus: stage C, vocal B, guitar C, drum B, atmosphere C
Outro: stage C or atmosphere B
```

For a 4-minute song, extend the second verse, bridge, and final chorus with extra vocal, guitar, stage,
and crowd variants. Keep the total usable Kling clip pool around 30 to 32 clips.
