# Testing Done Criteria

## Automated

- 기존 pytest 전체 통과
- 공연 프로필 데이터 참조 무결성 통과
- 결정론 및 파일별 변형 테스트 통과
- 대표 곡 생성 및 `validate` 통과
- 전체 `validate_output.py` 통과
- Python 구문 검사 통과
- 대표 출력의 실존 밴드명 검색 0건

## Human Q3/Q4

- Q3: 록, 감성곡, 전자음악 대표 결과를 나란히 비교한다.
- Q3: 이미지 구도와 영상 카메라 이동이 곡마다 구별되는지 확인한다.
- Q3: CapCut 전환 제안이 곡의 템포와 맞는지 확인한다.
- Q4: 얼굴·로고·정확한 의상·고유 소품·시그니처 무대 복제가 없는지 확인한다.

## Done

자동 검증 통과는 로컬 사용 가능 상태를 의미한다. 공개 업로드는 사람 검수를 별도로 통과해야 한다.
