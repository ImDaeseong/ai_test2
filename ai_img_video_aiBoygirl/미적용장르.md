## 새 프로파일 추가 방법

1. `genre_profiles.json`에 새 프로파일 추가:

```json
{
  "name": "jazz",
  "softening_type": "soft",
  "keywords": ["jazz", "bossa nova", "swing", "bebop", "smooth jazz"],
  "stage_energy": "intimate jazz live performance...",
  "lighting_style": "warm club spotlight...",
  "camera_style": "slow close-ups, steady jazz club push-in...",
  "special_effects": "soft smoke, warm ambient light particles...",
  "roles": {
    "vocal": "singing with jazz phrasing and warm vibrato",
    "vocal_action": "...",
    "guitar": "playing warm jazz chord voicings",
    "bass": "playing a walking jazz bass line",
    "drum": "playing restrained jazz brushwork on the snare",
    "crowd": "sitting quietly in the jazz club, nodding with the rhythm",
    "stage": "performing an intimate jazz club set"
  }
}
```

2. `run_all.bat` 실행 → 자동으로 전곡 재생성 + 검증

---

`validate_output.py`의 **[5] 미적용 장르 감지** 섹션이 input 폴더를 스캔해  
이 목록의 장르가 발견되면 자동으로 경고를 출력합니다.
