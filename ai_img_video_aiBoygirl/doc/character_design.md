## GPT 이미지 생성 시 사용 가이드

### 기준 이미지 첨부 순서
```
1. reference/{장르}.png    ← 해당 장르의 캐릭터 디자인 시트 (정면/측면/후면/표정 포함)
2. 01_master.png           ← 앞서 생성한 마스터 이미지 (있을 경우)
3. 07_stage.png            ← 앞서 생성한 무대 이미지 (개별 캐릭터 생성 시)
```

### AI Boy vs AI Girl 선택 방법
생성된 프롬프트에서 outfit 줄을 확인합니다:
```
AI Boy: {AI Boy 의상} / AI Girl: {AI Girl 의상}
```
→ 원하는 캐릭터 줄만 남기고 나머지 삭제 후 GPT에 붙여넣기.

### 캐릭터 일관성 유지 팁
- 새 대화를 시작할 때마다 reference PNG를 다시 첨부
- `02_style_lock_prompt.md` 내용을 대화 맨 앞에 붙여넣기
- 생성된 이미지(01_master.png)를 이후 모든 개별 이미지 생성 시 함께 첨부
